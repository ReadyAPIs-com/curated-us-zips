# Changelog

All notable changes to the `curated-us-zips` dataset are tracked here.

The dataset is regenerated from production whenever upstreams change.
The `main` branch always carries the latest snapshot; tagged releases
let you pin to a specific point-in-time if you need reproducibility.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

## [2026.05] - 2026-05-20

### Added
- Initial public release.
- 33,100 ZIP codes covering all 50 states + DC + 5 U.S. territories.
- 20 columns per row (see README "What's in the CSV" table).
- U.S. Census ACS 5-year (2022) demographic enrichment where available.

### Source snapshots
- U.S. Census Gazetteer: 2024 release.
- U.S. Census ACS: 5-year estimates, 2022 release.
- SimpleMaps US Zip Codes Basic: v1.86.
- IANA tz database: 2024a.
